## Dokumentasi Penelitian

Direktori `docs/` dipisah berdasarkan fase kerja supaya proposal awal tidak bercampur dengan skripsi final.

- `proposal-phase/`: seluruh artefak ujian proposal, termasuk naskah, slide, naskah presenter, dan QnA.
- `final-thesis/`: placeholder untuk dokumen skripsi final setelah fase proposal selesai.

Selama fase proposal aktif, semua target `make` mengarah ke `docs/proposal-phase/`.
