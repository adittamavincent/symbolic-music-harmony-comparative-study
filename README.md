# symbolic-music-harmony-comparative-study

**EVALUASI KEPATUHAN HARMONI FUNGSIONAL OUTPUT MODEL GENERATIF MUSIK SIMBOLIK: STUDI KOMPARATIF TEORI GUSTAV STRUBE**

Repository for ${RESEARCHER_NAME}'s undergraduate thesis proposal at the ${PROGRAM_STUDY}, ${FACULTY_NAME}, ${INSTITUTION_NAME}.

This project computationally evaluates the output of three distinct generations of symbolic AI music models (DeepBach, Coconet, and NotaGen) against Gustav Strube's functional harmony rules (e.g. prohibiting parallel 5ths/8ves, enforcing leading tone resolution). The algorithmic grading relies on Python and `music21`.

* `thesis/`: Contains the complete, rigorously formatted LaTeX source code for the proposal document.
* `README.md`: General overview.
